What is Danielcoin?
----------------

Danielcoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 1 hour block targets
 - subsidy halves in 500k blocks (A long time)
 - 1 million total coins
 - 1 coins per block
 - 24 blocks to retarget difficulty
